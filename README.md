# WFS文件系统

这是一个基于fuse的WFS文件系统

# 实现要求
| |要求 |是否完成 |
|----------------|--------------------------------| ------------------------------|
|1|文件目录里缺乏文件权限、文件主、时间日期等信息，请予以补全。 |完成|
|2|文件的大小目前最大只有一个磁盘块大小，请实现文件的大小可以是多块 |完成|
|3 |WFS目前只支持一级目录，即所以有的文件均在根目录下，请在WFS下实现多级目录|完成|
| 4|按照目前WFS目前的设计，文件的访问是一种顺序访问的方式，请改进WFS的数据访问方式为随机访问| 完成|

# 使用方法
1. git clone 地址
2. cp diskimg /tmp
3. make all
4. ./init_disk
5. ./WFS tt
